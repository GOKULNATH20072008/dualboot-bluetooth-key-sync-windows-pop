# Contributing



Thank you for considering contributing to this guide!



## How to Contribute



1. Fork the repository

2. Create a new branch (`git checkout -b improvement/amazing-feature`)

3. Make your changes

4. Commit your changes (`git commit -m 'Add amazing feature'`)

5. Push to the branch (`git push origin improvement/amazing-feature`)

6. Open a Pull Request



### Suggestions Welcome

- Hardware-specific notes for other laptops

- Additional troubleshooting tips

- Screenshots / GIFs

- Translations



---



**Note**: This is a community guide. Always back up your registry before running `bt-dualboot`.
